Examples to test the AIGER infrastructure.

The examples [01_model.aig](01_model.aig) and [06_witness.aig](06_witness.aig)
were generated from the corresponding `.aag` files in
[Certifaiger](https://github.com/Froleyks/certifaiger/tree/37f731e0ffa1d88ec5474c074d8088a161292d47/examples)
using [aiger](https://github.com/arminbiere/aiger)'s `aigtoaig`.
